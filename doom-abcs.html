SPC SPC - Find file in project

Resources

  • Github: +ivy/projectile-find-file

Summary

A more sensible counsel-projectile-find-file which will revert to counsel-find-file if invoked from $HOME, counsel-file-jump if invoked from a non-project, projectile-find-file if in a big project (more than ivy-sort-max-size files), or counsel-projectile-find-file otherwise.

The point of this is to avoid Emacs locking up indexing massive file trees.

SPC , - Switch workspace buffer
